https://nextjs.org/learn/foundations/from-react-to-nextjs/getting-started-with-nextjs


Versucht mal:
(Bitte zuerst in die repo navigieren und überprüfen,
dass ihr euch auf dem branch   siegenjs  befindet!)

erst dann:

        npm install react react-dom next


        npm install -g npm@9.6.6


(ka, ob ihr diese dependencies schon installiert habt,
wenn ja dann kriegt man möglicherweise ein Problem, 
aber das müssen wir uns dann individuell anschauen.)

es müsste der node_modules Ordner und der .next Ordner in der repo erscheinen. 


        npm run dev


der server sollte unter http://localhost:3000 zu erreichen sein.


Bitte am Dateienbaum und an der index.js Datei erstmal nix ändern. In pages könnt ihr neue files erstellen.
Ideen, Infos, Ressourcen und alles andere erstmal in den Ordner devs.

und:
git commit immer mit einer präzisen Beschreibung!
